Dashboard

If you want to see metrics graphics you need:

  • At first, check, have you list of needed metrics on prometheus. Go to the page Prometheus targets
  • Go to the Grafana page and authorize with admin credentials
  • Chose your metrics sources and create dashboard
  • On dashboard create panels which would draw graphics with your metrics
  • After panel creation, click on share button, go to Embed, change width=400 and height=200 to 100% in both and copy link
  • Press the button that below, paste the link and see result.
